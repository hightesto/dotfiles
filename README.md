my dotfiles for (black)arch linux

installation:
=
run install.sh

dependencies:
=
- i3-wm (with i3-gaps)
- i3blocks
- [i3-auto-layout](https://github.com/chmln/i3-auto-layout)
- optionally i3lock
- [picom](https://github.com/FT-Labs/picom) (AUR: picom-ftlabs-git)
- gnome-polkit
- brightnessctl
- networkmanager
- xss-lock (optional)
- oblogout
- xtrlock
- playerctl
